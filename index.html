FUNDAMENTOS HTML-5

¿Que es HTML?

HTML corresponde a las siglas en ingles de "HiperText Marck Lenguaje" y se puede traducir al español como "Lenguaje de Marcado de Hipertexto". Es un lenguaje de marcado diseñado para estructurar textos y presentarlos en forma de hipertexto, que es el formato estandara de las paginas web. Gracias a Internet y a los navegadores web, el HTML se ha convertido en uno de los formatos mas populares y faciles de aprendes que existen para la elaboracion de documentos para web. Es un lenguaje de sentencias-etiquetas- de las cuales me indican que operaciones se van a realizar con el texto o con los atributos que se esten manejando con ese sentencia-etiqueta-, la verdad estas secuencias son muy necesarias para el diseño de una pagina web ya que cada una de esas sentencias le indican a intert como esta compuesto la estructura de cualquier Web.

Etiqueta de HTML

Las etiquetas tienen una estrctura simple. Comienza por un caracter <, y terminan con un caracter >. Entre los caracteres <> esta el nombre de la etiqueta, y quizas algunos atributos dependiendo de la etiqueta. Su forma general es: 

     <_nombre de la etiqueta atributo1="valor1" ...>

Archivo HTML

Para hacer un archivo HTML correctamente: hay que identificarlo como archivo HTML encerrandolo entre etiquetas <!doctype html> al principio. Despues se pone <html lang="es"> para indentificar al documento HTML que esta en lenguaje español, esta etiqueta contiene un <head> que te permite almacenar cierta informacion sobre el documento en si, tambien contiene <body>,dentro de esta tiene todo el contenido de la pagina que se ve a travez del navegador web, despues de estas dos etiquetas se cierra todo con </html>. Su estructura seria asi: 

    
                 <html lang="es">      Inicio
                      <head>           Cabecera
                      </head>
                      <body>           Cuerpo
                      </body>
                 </html>               Fin

Metadatos

Las etiquetas de matadatos se deben añadir entre las etiquetas <head> head> del codigo. Estas etiquetas contienen la informacion que se muestran en las paginas de resultados de los diferentes buscadores. Los metadatos html mas habituales e importantes son: 

         
    
            
  • Title:
    •      <title>     </title> El titulo de la pagina que sale en el buscador.
            
  • Charset
    •     <meta charset="utf-8"> Especifica la codificacion de caracteres para el documento.
            
  • Name
    •       <meta name="author" content="Ray>" Ponemos el nombre del autor de la pagina.

Articulo vs Seccion

Una seccion usa las etiquetas <section </section>, esta pensada para agrupar los apartados del contenido del documento. Una seccion puede contener varias secciones dentro tambien. Un articulo usa las etiquetas <article> </article>, esta pensado para agrupar el contenido de la pagina que forma una unidad en si misma desde el punto de vista tematico, puede contener mas etiquetas de articulo en su interior.Un elemento <section> puede contener elementos <article> y viceversa. 

             <article>                       <article>
                <section>                       <section>
                </section>                          <article>
                <section>                           </article>
              </article>                        </section>
                                             </article>

Secciones

Las secciones son las partes tematicas en las que se puede dividir el contenido de una pagina web. Estas etiquetas van entre las etiquetas <body> </body> y son:

  • Encabezamiento:
    • La etiqueta <header> define la cabecera de una pagina o seccion. Usualmente contiene un logotipo,el titulo del sitio Web y una tabla de navegacion de contenidos.
  • Navegacion:
    • La etiqueta <nav> define una seccion que solamente contiene enlaces de navegacion.
  • Titulos:
    • Las etiquetas <h1>,<h2>.<h3>,<h4>, <h5> y <h6> implementan seis niveles de cabeceras de documentos, <h1> es la de mayor y <h6> es la de menor importancia.
  • Lateral:
    • La etiqueta <aside> define algunos contenidos vagamente relacionados con el resto del contenido de la pagina.
  • Pie:
    • La etiqueta <footer> define el pie de una pagina o seccion, suele contener un mensaje de derechos de autoria,enlaces a informacion lega o direcciones para dar informacion.
  • Direccion:
    • La etiqueta <adress> define una seccion que contiene informacion de contacto. 
          <header>
              <h1>     </h1>
              <h2>     </h2>
          </header>
                  <nav>
                  </nav>
          <footer>
                  <adress>
                  </adress>
          </footer>

Agrupaciones

Hay diferentes tipos de agrupaciones, pero las mas importantes son:

  • Parrafos:
    • Las etiquetas <p> <p> definen una parte que debe mostrarse como un parrafo.
  • Texto perfomateado
    • Las etiquetas <pre> <pre> indican que su contenido esta preformateado y que este formato debe ser preservado.
  • Listas:
    1. Ordered List: <ol>
      2. Define una lista ordenada de articulos.
    2. Unordered List: <ul>
      3. Define una lista de articulos sin orden.
    3. List Item: <li>
      4. Define un articulo de una lista.
    4. Definition List: <dl>
      5. Define una lista de definiciones,es decir, una lista de terminos y sus deficiones asociadas.
    5. Definition Term: <dt>
      6. Representa un termino definido por el siguiente <dd>.
    6. Definition Data: <dd>
      7. Representa la deficion de los terminos listados antes que el. 
                     Lista ordenada:                          Lista desordenada:                          Lista de definicion:
                  <ol>                                   <ul>                                         <dl>
                  <li> Primer elemento </li>             <li> Primer elemento </li>                   <dt>Termino1
                  <li> Segundo elemento </li>            <li> Segundo elemento </li>                  <dd>Definicion del Termino1
                  <li> Tercer elemento </li>             <li> Primer elemento </li>                   <dt>Termino2
                  </ol>                                  </ul>                                           <dt>Definicion del Termino2
                                                                                                      </dl>
  • Linea horizontal:
    • La etiqueta <hr> representa una linea que sirve para separar parrafos de una seccion o articulo o cualquier contenido.

Semantica

Hay diferentes tipos de elementos semanticos, pero los mas importantes son:

  • Resaltado
    1. Bastardilla-Italic <i>:
      2. Pone en cursiva una palabra o un parrafo que este dentro de <i> </i>.
    2. Negrita-Bold <b>:
      3. Pone en negrita una palabra o un parrafo que este dentro de <b> </b>.
    3. Subrayada-Underlined <u>:
      4. Subraya una palabra o un parrafo que este dentro de <u> </u>.
    4. Stress Emphasis <em>:
      5. Pone enfasis o recalca una parte del texto que este entre <em> </em>.
    5. Strong <strong>:
      6. Parte importante,seria o urgente que este dentro de <strong> <strong>.
    6. Small <small>:
      7. Representa la "letra pequeña" (de un contrato..), tiene que estar entre <small> </small>.
    7. S <s>:
      8. Representa contenido irrelevante u obsoleto, tiene que estar entre <s> </s>
  • Salto de linea
    1. Break <br>:
      2. Representa un salto de lina despues de que pongas solo <br>
    2. Word Break <wrb>:
      3. Representa una oportunidad de salto de linea, es decir, un punto sugerido de envoltura donde el texto de multiples lineas puede ser dividido para mejorar su legibilidad.
  • Hiperenlace
    • Sirven para acceder desde una pagina a otra o a otro recurso disponible.
    1. Externo
      2. El atributo href contiene la informacion de donde se encuentra el recurso. En este caso en un dominio externo. 
      <a href="http://wwww.undominio.com" target="_blank">Undominio</a>
    2. Local
      3. En este caso href apunta a una pagina html que eseta en el mismo directorio que el documento que contiene el enlace. 
       <a href="pagina2.html" target="_blank">Ver esto </a>
    3. Interno
      4. En este caso href apunta a una posicion dentro del documento en el que esta el enlace. 
       <a href="#ancla1">Ir a..<a>

Estamos usando el atributo target que indica donde se cargara el documento o recurso al que hacemos referencia y el valor _blank indica que cargemos el recurso en una nueva ventana del navegador.

Contenido Encajado:Imagenes

Vamos a insertar imagenes dentro de una pagina web, para poner una imagen el codigo es: 

          <img src="imagen.jpg">

La etiqueta <img> tiene varias propiedas. El ancho y alto (width,height) se coloca en pixeles y deben corresponder al tamaño original de la imagen para que no se distorsione. 

    <img src="imagen.jpg"  width="10" height="10">

Tambien podemos usar un mapa de imagenes, es decir, podemos hacer que diferentes partes de una imagen tengan enlaces que nos permitan a ir a diferentes destinos.

  • El tag map:
    • Aqui tenemos que poner la etiqueta <map name="ejemplo1"> </map>, aqui añadimos el nombre del mapa dentro del mapa van los demas atributos.
  • Atributo area:
    • El mapa de imagenes lleva un area dentro de la cual podremos pinchar para dirigirnos al destino que hayamos preestablecido <area>, dentro de area iran otros atributos.
  • Atributo shape:
    • Estre atributo va dentro de area y nos indica que tipo de area es.
    1. shape="rect":
      2. Este tipo de area crea un mapa de imagen rectangular. Para definir la zona que incluira este mapa deberemos definir la esquina superior del area y la esquina inferior derecha.
    2. shape="circle":
      3. Este tipo de area crea un mapa de imagen circular. Nosotros solo debemos indicarle el centro de la circuferencia y el radio del mismo.
    3. shape="poly":
      4. Este tipo de area es la mas versatil de todas, ya que nos permite crear una zona personalizada. Debemos indicar los diferentes puntos de poligono y de una forma ordenada.
  • Atributo coords:
    • Aqui escribimos las coordenadas del area que vayamos a usar se escribe: coords="x,x,x,x", entiendo que cada "x" representa un punto de la imagen.
    1. rect:
      2. "x-izq,y-sup,x-dcha, y-inf"
    2. circle
      3. "x-centro,y-centro,radio"
    3. poly
      4. "x1,y2,x2,y2,....,xN,yN"
  • Usemap:
    • Aqui indicamos que imagen es la que llevara el mapa, este atributo se escribe: usemap="#x", siendo x el nombre del mapa de la imagen. 
        <img usemap="#mapa">
        <map name="mapa">
          <area shape="    " coords="x,x,x,x">
        </map>

Tablas

Las tablas en HTML te permiten mostrar un arreglo de celtas de datos:

  1. Las tablas se definen con la contenedora <table>.
  2. La etiqueta <table> contiene reglones de celdas, definidos con la contenedora <tr>.
  3. Cada etiqueta <tr> contiene celdas de datos, definidos con la contenedora <td>.<
  4. Cada celda de datos contiene lo que quieras poner como imagenes, listas, incluso otras tablas.

Tambien tiene atributos y son:

  1. El ataributo border, se usa para mostar un borde al rededos y entre las celdas de la tabla (en table).
  2. El atributo rowspan=x, es una expansion en la columna, siendo x el numero de casillas que queremos ocupar (en td y th)
  3. El atributos colspand=x, es una expasion en la fila, siendo x el numero de casillas que queremos ocupar (en td y th). 
          <table border>
          <tr>
            <td rowspan=2>oeste </td>
            <td>noreste</td>
          </tr>
          <tr>
            <td>sureste</td>
          <tr>
          </table
